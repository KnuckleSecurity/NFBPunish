# NFBPunish
Instagram unfollower bot
<br>
![Desktop View](misc/pic1.png)

## Prerequisities
* Python3
* Pip3
* Mozilla Firefox
## Installiaton

```
git clone https://github.com/krygeNNN/NFBPunish
```
For Windows:--> 
Run `setup.bat`

For Linux:->> 
Run `setup.sh`
<br>
Run `sudo pacman -S tk` for Arch, or `sudo apt install tk` for Debian.
